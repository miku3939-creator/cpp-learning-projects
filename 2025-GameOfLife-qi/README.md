# 生命游戏 (Conway's Game of Life)

## 项目描述
基于C++和EasyX图形库实现的康威生命游戏，这是一个经典的细胞自动机模拟程序。程序提供了完整的图形化界面，支持自定义游戏规则和实时交互控制，展现了细胞世界的演化规律。

## 功能特性
- 鼠标左键绘制细胞，右键擦除细胞
- 支持B3/S23、B36/S23等多种生命游戏规则
- 开始、暂停、重置、速度调节功能
- 实时显示代数、种群数量、游戏模式
- 无闪烁的流畅动画效果
- 网格显示/隐藏切换

## 运行说明
### 直接运行
双击 `bin/LifeGame.exe` 即可启动程序，无需安装任何额外依赖。

### 程序操作指南
1. **初始设置**：
   - 程序启动后处于设置模式
   - 使用鼠标左键在网格上绘制细胞
   - 使用鼠标右键擦除细胞

2. **运行控制**：
   - 点击"开始演化"按钮开始自动演化
   - 点击"暂停"按钮暂停演化过程
   - 点击"重置"按钮清空所有细胞

3. **规则设置**：
   - 点击"规则设置"按钮自定义演化规则
   - 输入格式：`B.../S...`（例如：B3/S23）
   - B代表出生条件，S代表存活条件

4. **其他功能**：
   - 使用滑块调节演化速度
   - 点击"隐藏网格/显示网格"切换网格显示
